# Doctor-Patient-Appointment---python-django
Doctor Appointment Booking System is a Python (Django) web application that allows patients to book doctor appointments online.  Tech Stack: HTML, JavaScript, Python (Django), MySQL | OS: Windows 7+.
#Project Overview

In traditional systems, patients must physically visit hospitals, wait in queues, and face uncertainty if appointments are canceled.

This project solves those problems by enabling:

Online appointment booking
Real-time availability checking
Patient & doctor management
Easy communication between users

#Features
👨‍💼 Admin Module
Add, update, delete doctors
View patient details & history
Manage appointments
View feedback from patients
👨‍⚕️ Doctor Module
View appointments
Manage profile
Access patient history
Add treatment details
Patient Module
Register & login
Book appointments (date & time slots)
View booking history
Cancel appointments
Search doctors (name, specialization, location)
Give feedback

#Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Python (Django Framework)
Database: MySQL
Environment: Windows / XAMPP
